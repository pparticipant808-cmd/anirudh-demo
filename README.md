# Playwright OrangeHRM Automation

This project automates the testing of the **OrangeHRM login page** using the **Playwright** testing framework. It verifies the login functionality, ensuring both valid and invalid login scenarios are correctly handled.

## Description

This project uses **Playwright** for automating browser interactions and testing the login functionality of the **OrangeHRM demo website**. The tests ensure that the login page works as expected across different browsers (Chromium, Firefox, and WebKit).

### Purpose:
- Automate functional testing of the **OrangeHRM login page**.
- Test both **valid** and **invalid login scenarios**.
- Use **Playwright** for cross-browser testing to ensure that the login functionality works consistently across Chromium, Firefox, and WebKit browsers.

## Features

- **Login Test Automation**: Automates login and verifies behavior for valid and invalid credentials.
- **Cross-Browser Testing**: Supports testing in **Chromium**, **Firefox**, and **WebKit**.
- **Playwright Integration**: Uses **Playwright** to manage browsers and perform automated tests.
